Lingomon Privacy Policy

Last updated: January 26, 2026

Data Collection

Lingomon collects and stores the following data:

  • Vocabulary: Words you select and catch from web pages.
  • Context: The specific sentence where you found the word (saved locally to help you remember usage).
  • Definitions: Word definitions and scientific data fetched from various external APIs.
  • Statistics: Website domains where words were caught (saved locally to track unique site visits), your achievements, streaks, and badges.
  • Profile: Your username, avatar preference, and battle team configuration.
  • Language Preference: Whether you are using English or Korean mode.

Data Storage

  • Local Storage: All gameplay data (WordDex, achievements, stats) is primarily stored locally on your device using Chrome's chrome.storage.local API.
  • Cloud Sync (Optional): If you choose to enable multiplayer features, your Profile (username, avatar, battle team) and limited stats (win/loss record) are stored in our secure cloud database (Supabase).
  • Authentication: We use an anonymous User ID generated locally. We do not collect email addresses or passwords.

Third-Party Services & APIs

Lingomon connects to the following external services to fetch definitions and game data. No personal browsing history is sent to these services. Only the specific word you catch is transmitted.

  • Supabase (supabase.com): Stores public profiles and matchmaking data for the Battle Arena.
  • Dictionary API (api.dictionaryapi.dev): Provides English word definitions.
  • Datamuse API (api.datamuse.com): Provides word frequency and related terms.
  • PubChem (NIH) (pubchem.ncbi.nlm.nih.gov): Retrieves chemical data (formulas, molecular weight) for the Chemistry module.
  • Solar System OpenData (api.le-systeme-solaire.net): Retrieves astronomical data for planets and stars.
  • GBIF (api.gbif.org): Retrieves biological taxonomy data for species.
  • StackExchange API (api.stackexchange.com): Identifies technical/IT terms and tags.
  • Korean Learners' Dictionary API (krdict.korean.go.kr): Provides Korean definitions.
  • DiceBear API (api.dicebear.com): Generates robot avatars based on your User ID.

Your Rights

  • You can export your local data at any time (via Chrome Storage).
  • You can delete your account and cloud data by contacting us.
  • You can use the extension without logging in, but cloud backup and multiplayer features will be unavailable.
